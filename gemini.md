# app-multiplay Analysis

## Overview
A cross-platform mobile application built with Expo (React Native). It appears to be a multi-tabbed application supporting Android, iOS, and Web platforms.

## Tech Stack
- **Framework**: Expo 54, React Native 0.81.
- **Navigation**: React Navigation 7 (Bottom Tabs, Material Top Tabs).
- **State/Storage**: Async Storage.
- **Animations**: React Native Reanimated.
- **Icons**: Lucide React Native.

## Structure
- `App.js`: Main entry point.
- `app.json`: Expo configuration.
- `src/`: Application source code.
- `web/`: Web-specific assets or config.

## Key Features
- **Cross-Platform**: Runs on Android, iOS, and Web from a single codebase.
- **Navigation**: Complex navigation structure using tabs.
- **Native Features**: Safe area context, gesture handling.

## Setup & Run
1. Install dependencies: `npm install`
2. Start Expo server: `npx expo start`
   - Press `a` for Android, `i` for iOS, or `w` for Web.
